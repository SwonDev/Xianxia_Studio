# v0.2.6 — ACE-Step v1.5 removed from the pipeline. On 8 GB VRAM it
# requires cpu_offload=True (README oficial) which contradicts the
# project's GPU-only rule. On Windows the offload-disabled path hangs
# indefinitely via WDDM thrash. Upstream repo has no release tag and
# the inference loop is unmaintained. We rely solely on MusicGen plus
# the local library as the final fallback.

# 30 s hard cap per pass; long-form is chunked + acrossfade'd in
# routes/music.py. Good general cinematic quality.
